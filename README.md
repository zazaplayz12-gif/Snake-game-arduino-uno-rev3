# Snake-game-arduino-uno-rev3
Retro Snake game for Arduino Uno R3 using an 8x8 MAX7219 LED Matrix and analog joystick. Features a 1-dot startup, non-blocking millis() engine for smooth 4-way steering, smart food spawning, and click-to-pause via the joystick button.   Wiring: Matrix (DIN:12, CS:11, CLK:10) | Joystick (VRx:A0, VRy:A1, SW:D2). Uses LedControl library.
